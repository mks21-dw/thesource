  (lambda (g)
  (and (> (foldl + 0 g) 50)
       (< (foldl (lambda (x y) (+ x 1)) 0 g) 5))))

Programming question:
Write a program that defines a variable "booksRead" as a list with 12 elements each with a value between 0 and 16.
Then use this variable to define "monthPages" that, using function "pageCount", creates a new list with the pages read that month (odd # books = 300 pgs/book, for even# = 400 pgs/book)
Then use this to define "yearTotal" by writing a function that recursivly determines how many pages were read during the year.
Skill: 5, 6, 7, 10, 11
Description


(define booksRead (build-list 12 (lambda (x) (random 0 16))))
(define pageCount
  (lambda (a) (if (null? a)
                  '()
                  (if (even? (first a))
                      (cons (* (first a) 400) (pageCount (rest a)))
                      (cons (* (first a) 300) (pageCount (rest a)))))))
(define monthPages (pageCount booksRead))
(define pageTotal (lambda (b) (if (null? b)
                                  0
                                  (+ (first b) (pageTotal (rest b))))))
(define yearTotal (pageTotal monthPages))
booksRead
monthPages
yearTotal

Question 52:

Programming question:
Make a function that takes in two lists of numbers
with the same # of elements. Then, take the first value
of one list and the last value of the second
list and computes the sum of their squares.
Then, repeat the process until there are no
elements left, and output the values as a list.

Skill: 11, 3

(define helper
  (lambda (l1 l2)
     (if (null? l1)
      l2
      (cons (first l1) (helper (rest l1) l2)))))
(define swap
  (lambda (a)
    (if (null? a)
      empty
      (helper (swap (rest a)) (list (first a)))))
(define (sumsq l1 l2)
  (if (null? l1)
      empty
      (cons
       (+ (* (first l1) (first l1))
          (* (first (swap l2)) (first (swap l2))))
       (sumsq (rest l1) (swap (rest (swap l2)))))))
